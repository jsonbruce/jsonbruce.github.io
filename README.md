# TechStack
Hexo + GitHub Pages


# Usage
1. Clone Source Code of Blog
```
git clone git@github.com:jsonbruce/jsonbruce.github.io.git
git checkout source
```

2. Edit Articles

3. Post
```
hexo clean
hexo d -g
```
